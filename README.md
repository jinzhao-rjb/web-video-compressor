# WebAssembly 视频压缩工具

一个基于 WebAssembly 和 FFmpeg.wasm 构建的在线视频压缩工具，支持视频预览、压缩设置调整和压缩前后对比功能。

## 功能特性

- 📁 **视频上传**：支持拖拽或点击上传视频文件
- 🎬 **视频预览**：实时预览原始视频和压缩后视频
- ⚙️ **压缩设置**：
  - 可调节压缩质量（1-100%）
  - 可选择分辨率缩放比例
- 📊 **压缩对比**：
  - 显示压缩前后文件大小对比
  - 计算压缩率和节省空间
  - 支持双窗口对比播放
- 💾 **快速下载**：一键下载压缩后的视频文件

## 技术栈

- **前端框架**：原生 HTML, CSS, JavaScript
- **WebAssembly**：FFmpeg.wasm
- **样式**：Tailwind CSS + 自定义样式
- **图标**：Font Awesome

## 快速开始

### 方法一：直接打开

1. 下载或克隆项目到本地
2. 在项目根目录下找到 `index.html` 文件
3. 直接用浏览器打开该文件即可使用

### 方法二：本地服务器

1. 确保安装了 Python
2. 在项目根目录下运行命令：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 使用说明

1. **上传视频**：点击或拖拽视频文件到上传区域
2. **调整设置**：
   - 使用滑块调整压缩质量
   - 选择合适的分辨率缩放比例
3. **开始压缩**：点击"开始压缩"按钮
4. **查看结果**：
   - 压缩完成后会自动播放压缩后的视频
   - 查看压缩前后的文件大小对比
   - 在对比区域同时播放原始视频和压缩后视频
5. **下载视频**：点击"下载压缩视频"按钮保存文件

## 支持的视频格式

- MP4
- WebM
- 其他 FFmpeg 支持的视频格式

## 注意事项

1. 视频文件大小建议不超过 100MB，过大的文件可能导致浏览器崩溃
2. 压缩过程可能需要几分钟时间，取决于视频大小和浏览器性能
3. 首次使用时会加载 FFmpeg.wasm 核心文件，可能需要一些时间
4. 建议使用 Chrome 或 Firefox 浏览器以获得最佳体验
